# Debian 13 (Native System Integration 100%)
## dependencies
- python3-pyside6.qtcore 
- python3-pyside6.qtwidgets
## install
- sudo apt install python3-pyside6.qtcore python3-pyside6.qtwidgets

# Ubuntu 24.04 (Noble Numbat)
## Issue
- Package 'python3-pyside6' is missing from official repositories (Universe).
## Solution
- Must use pip within a virtual environment or use uv.
## install
1. Install pip and venv:
   sudo apt install python3-pip python3-venv
2. Create venv and install:
   python3 -m venv .venv
   source .venv/bin/activate
   pip install pyside6
